## How to install

## Step One: Installing [MelonLoader](https://melonwiki.xyz/)
   1) Quit Sledders if running
   2) Download melon loader [here](https://melonwiki.xyz/#/?id=automated-installation)
   3) Run the exe and select sledders and install
   4) Start sledders

## Step Two: Installing the Menu
  1) Quit Sledders again
  2) Download the __.dll__ file [here](https://github.com/Ronnie-Reagan/Alpine-Tuning/releases/download/2025.12.20/Alpine-Tuning.dll)
  3) Move the __.dll__ file to `C:\Program Files (x86)\Steam\steamapps\common\Sledders\Mods`
     - Your location may be different;
     - In Steam **Library** -> Right Click **Sledders** -> **Manage** -> **Browse local files** -> [In Popup] Select the **Mods** folder

## Step Three: How to tune a sled
  1) Load **Sledders**
  2) Start a local game or join a lobby
  3) Review the menu for features then press F4 to hide it
  4) Pause the game and change to your desired sled - unpausing when selected
  5) Pause the game again and Press F4 to open the menu
  6) Make changes to the sled in the menu
  7) Enable `Auto Apply` then press `Apply` in the menu
  8) Pause the game and change to a __differet__ sled - unpausing when selected
  9) Pause the game and switch back to the **Tuned sled**
